# Инструкция по настройке синхронизации 1С для владельца

Если вы делаете настройку сами, следуйте этим шагам. Эта настройка подходит для 1С:УНФ, 1С:Управление торговлей и 1С:Розница.

## Шаг 1: Где найти настройки в 1С
1. Откройте **1С**.
2. Перейдите в раздел **CRM и продажи** (или **Компания** / **Настройки**).
3. Найдите пункт **Интеграция с сайтом** или **Обмен с сайтами**.
4. Нажмите кнопку **Создать** (или "Настроить обмен с сайтом").

## Шаг 2: Что вписать в поля
В открывшемся окне выберите "Выгрузка на сайт" и заполните:

*   **Адрес сайта:** `https://[ваш-адрес-проекта].replit.app/api/1c-exchange`
*   **Имя пользователя:** `admin`
*   **Пароль:** `bmg-secret-123`

## Шаг 3: Проверка связи
1. Нажмите кнопку **Проверить соединение**. 
2. Если всё верно, 1С напишет: *"Соединение с сайтом успешно установлено"*.

## Шаг 4: Настройка выгрузки
1. Перейдите на вкладку **Режим обмена**.
2. Поставьте галочку **Выгружать товары**.
3. В таблице ниже выберите те категории товаров, которые вы хотите видеть на сайте.
4. Нажмите **Записать и закрыть**.

## Шаг 5: Первый запуск
1. В списке настроек выберите созданную интеграцию.
2. Нажмите кнопку **Выполнить обмен**.
3. Дождитесь завершения. После этого товары из 1С появятся в базе данных сайта.

---
**Важно:** 
*   Сайт теперь понимает стандартный формат 1С (CommerceML).
*   Если у вас возникнут сложности с конкретной кнопкой — просто напишите мне название вашей версии 1С, и я подскажу точнее.
